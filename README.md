# HyacinthSat 官方网站

本仓库为 HyacinthSat（风信子卫星项目）的官方网站前端代码，用于展示项目信息、核心特性和联系方式。

## 目录结构

```
├── README.md        -------- 项目说明  
├── LICENSE          -------- 开源许可（MIT）  
├── .gitignore       -------- Git 忽略规则  
├── index.html       -------- 网站主页  
└── src/             -------- 源代码目录
    ├── css/         -------- 样式文件
    │   └── style.css
    ├── js/          -------- JavaScript 文件
    │   └── script.js
    └── assets/      -------- 静态资源（图片、图标等）
```

## 功能特性

本网站具备以下核心功能：

- **响应式设计**：适配各种设备尺寸（桌面、平板、手机）
- **现代化界面**：采用渐变色彩和流畅动画
- **平滑滚动**：优化的页面导航体验
- **表单交互**：联系表单包含验证和提交反馈
- **移动端菜单**：支持移动设备的汉堡菜单
- **动画效果**：元素滚动进入视口时的淡入动画

## 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/HyacinthSat/hyacinthwf.git
cd hyacinthwf
```

2. 在浏览器中打开 `index.html` 文件即可查看网站

3. 如需本地服务器（推荐），可以使用以下任一方法：

使用 Python:
```bash
python -m http.server 8000
```

使用 Node.js http-server:
```bash
npx http-server
```

然后在浏览器访问 `http://localhost:8000`

## 技术栈

- **HTML5**：语义化标签，良好的结构
- **CSS3**：现代样式，CSS Grid 和 Flexbox 布局
- **Vanilla JavaScript**：无框架依赖，原生 JS 实现
- **响应式设计**：Mobile-first 设计理念
